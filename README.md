# 🚀 ProQuant AI – Stock Forecasting Dashboard

An advanced **AI-powered financial analytics dashboard** that predicts stock prices and provides intelligent insights using **Facebook Prophet**, **Streamlit**, and **Plotly**.

Designed to simulate a **real-world trading terminal**, this project combines **machine learning + data visualization + UI engineering** into a single system.

---

## 📊 Key Features

* 📈 **Time Series Forecasting** using Prophet
* ⚡ **Real-time stock data** via Yahoo Finance API
* 📉 **Momentum Analysis (RSI)** for market condition detection
* 🎯 **AI-based future price prediction (Target Price)**
* 📊 **Interactive charts** (Plotly with zoom & hover)
* 🧠 **AI Intelligence Report** (trend + insights)
* 🎨 **Premium Dark UI Dashboard (Glassmorphism style)**

---

## 🧠 Model Overview

The forecasting engine is built using **Prophet**, which models time series data as:

> Trend + Seasonality + Noise

### ✔ Model Capabilities:

* Automatic trend detection
* Handles missing data
* Detects seasonal patterns
* Supports change-point detection

---

## 📊 Model Performance

After tuning the model:

* ✅ **MAE:** ~4.69
* ✅ **RMSE:** ~6.07

✔ Indicates improved prediction accuracy and stability
